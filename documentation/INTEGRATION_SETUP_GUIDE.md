# Integration Setup Guide

## Twilio / WhatsApp
To connect Twilio, users must provide:
1. **Account SID**: Found on the Twilio Console homepage.
2. **Auth Token**: Found next to the Account SID.
3. **WhatsApp Number**: In the format `+14155238886`.

*Note for webhook routing:* The Twilio webhook pointing to `/api/whatsapp/webhook` will automatically parse the `To` number and route the incoming message to the correct `Business` document matching that number.

## Google Business Profile
The platform requires:
1. **Google Place ID**: Extracted using the Google Maps API.
2. **Review Link**: The shortlink generated by GBP (e.g., `g.page/r/...`).

## AI Configuration
The `Business` model stores the prompt architecture. When an incoming WhatsApp message hits the AI, the context window is injected with:
- `business.name`
- `business.aiSettings.tone`
- `business.aiSettings.salesPrompt`
This ensures the AI speaks accurately on behalf of the tenant.
